Sobre o Projeto

O Universo C é um Recurso Educacional Aberto (REA) criado para a disciplina de Fundamentos da Programação. Nosso foco é tornar o aprendizado da linguagem C mais acessível, interativo e visual.

Acreditamos que, com conteúdo objetivo e prática constante, qualquer pessoa pode dominar os pilares da programação e construir uma base sólida para uma carreira em tecnologia.

Módulos do Curso

Módulo 1: Fundamentos da Linguagem

Variáveis, tipos de dados, operadores e entrada/saída de informações (printf/scanf).

Módulo 2: Introdução e Ambiente

Conceitos iniciais, configuração do ambiente (IDE) e o primeiro "Olá, Mundo!".

Módulo 3: Estruturas Condicionais

Controle o fluxo do seu código com as estruturas if, else e switch-case.

Módulo 4: Estruturas de Repetição

Execute tarefas repetitivas de forma eficiente com os laços for, while e do-while.

Módulo 5: Vetores (Arrays)

Aprenda a armazenar e manipular coleções de dados de um mesmo tipo.

Módulo 6: Matrizes

Expanda seu conhecimento de arrays para trabalhar com múltiplas dimensões (tabelas).

Módulo 7: Strings

Introdução ao conceito de strings em C e funções básicas de manipulação.

Módulo 8: Funções

Modularize seu código com funções, parâmetros e valores de retorno.

Módulo 9: Conclusão

Parabéns por completar a jornada! Esperamos que esta base sólida impulsione seus futuros projetos.
